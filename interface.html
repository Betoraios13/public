Interface

Para um sistema de interações ser desenvolvido vamos utilizar o método interface. Assim, evitaremos grandes linhas de códigos com if e else.

UsingSystem.Collections;
UsingSystem.Collections.Generic;
UsingUnityEngine;

public classinterface: MonoBehaviuor
{
private voidStart()
{

}


private voidUpdate()
{

}
}

Apague toda a classe e métodos originais do script(void start, void update, MonoBehaviour). Ademais, você irá escrever como na próxima imagem demonstra:

UsingSystem.Collections;
UsingSystem.Collections.Generic;
UsingUnityEngine;

public interface interacao
{
voidinteragir();
}

Agora você deve acessar o código do objeto que deseja interagir na cena , por exemplo:

UsingSystem.Collections;
UsingSystem.Collections.Generic;
UsingUnityEngine;

public classbau : MonoBehaviuor
{
public voidAbrir()
{
//sua lógica para o objeto
}
}

em seguida adicione as seguintes alterações:

UsingSystem.Collections;
UsingSystem.Collections.Generic;
UsingUnityEngine;

public classbau : MonoBehaviuor ,interacao
{
public voidinteragir()
{
Abrir();
}

public voidAbrir()
{
//sua lógica para o objeto
}
}

agora no script do player você colocará isto:

UsingSystem.Collections;
UsingSystem.Collections.Generic;
UsingUnityEngine;

public classplayer : MonoBehaviuor
{
private void OnCollisionEnter(Collision collision)
{
interacao obj = collision.transform.GetComponent<interacao>();

if(obj != null) return;

obj.interagir();
}
}

Esta lógica pode ser adaptada para triggers, collision, Raycast e etc.

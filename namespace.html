NameSpace

Esta função é utilizada para o agrupamento de classes em seus códigos servindo também para organização

não é algo muito usado no desenvolvimento de jogos com unity. Mas, com o nameSpace você poderá ter scripts com nomes iguais mas com funcionalidades diferentes 

                   using System.Collections;
                   using UnityEngine;

                   nameSpace Player.Move
                   {
                        public class movimento : MonoBehaviour
                        {
                            // Seu código
                        }
                   }

neste script vemos a class movimento dentro de uma nameSpace: Player.Move

                   using System.Collections;
                   using UnityEngine;

                   nameSpace Enemy.Move
                   {
                        public class movimento : MonoBehaviour
                        {
                            // Seu código
                        }
                   }

e neste outro script temos a mesma class mas um nome space diferente

OBS: Os scripts criados para o player e o inimigo devem ficar em pastas distintas obrigatoriamente, mas o nameSpace evita erros no seu editor de código

OBS: Um nomeSpace não pode conter duas classes de nomes iguais, mas vc pode ter duas classes iguais em NameSpaces diferentes
